🐣 День 2: Строки в Python

📖 Словари в Python: dict

Словарь (dictionary) — это неупорядоченная (до Python 3.6) или упорядоченная по вставке (Python 3.7+) коллекция пар ключ: значение. Аналогия из жизни: телефонная книга, где имя — ключ, а номер — значение. Или инвентарь в игре, где название предмета хранит количество.

🔑 Главное правило:
• Ключи должны быть неизменяемыми (строки, числа, кортежи).
• Ключи уникальны (повтор перезапишет старое значение).
• Значения могут быть любого типа (числа, строки, списки, другие словари).

🔨 Создание словаря

# Литерал (самый частый способ) prices = {"яблоко": 50, "хлеб": 80, "молоко": 95} # Пустой словарь empty = {} # Через функцию dict() и список кортежей config = dict([("theme", "dark"), ("lang", "ru")])

📥 Чтение и изменение данных

prices = {"яблоко": 50, "хлеб": 80} # Чтение по ключу print(prices["яблоко"]) # 50 # Изменение существующего значения prices["яблоко"] = 55 # Добавление новой пары prices["банан"] = 40 # ⚠️ Ошибка KeyError при несуществующем ключе! print(prices["груша"]) # упадёт с ошибкой

Чтобы избежать KeyError, используйте оператор in или метод .get().

🛡️ Безопасное чтение: .get()

prices = {"яблоко": 55, "хлеб": 80} # Вернёт значение, если ключ есть print(prices.get("хлеб")) # 80 # Вернёт None, если ключа нет print(prices.get("груша")) # None # Вернёт значение по умолчанию print(prices.get("груша", 0)) # 0

.get() не меняет словарь и никогда не вызывает ошибку. Идеально для подсчётов и агрегаций.

🔄 Перебор словаря

menu = {"эспрессо": 80, "латте": 150, "какао": 100} # 1. Только ключи (по умолчанию) for item in menu: print(item) # 2. Ключи и значения одновременно for drink, price in menu.items(): print(f"{drink} стоит {price} руб.") # 3. Только значения for p in menu.values(): print(p)

💡 .items() возвращает кортежи (ключ, значение). Распаковка в цикле — стандарт Python.

🗑️ Удаление элементов

data = {"a": 1, "b": 2, "c": 3} # del: удаляет по ключу (ничего не возвращает) del data["a"] # .pop(): удаляет и возвращает значение removed = data.pop("b") # removed = 2 # .popitem(): удаляет последнюю добавленную пару (LIFO) last_key, last_val = data.popitem() # .clear(): очищает весь словарь data.clear()

💡 Где словари используются на практике:
  • Конфигурации и настройки программ (settings.json → dict)
  • Кэширование результатов вычислений (memoization)
  • Подсчёт частоты слов, голосов, товаров
  • API-ответы (JSON автоматически парсится в словарь)
  • Игровые инвентари, профили пользователей, базы данных в памяти

Задание 1: Инвентарь искателя приключений

Цель

Игрок исследует подземелья и собирает предметы. Напиши программу, которая принимает строку с перечнем найденных предметов и вычисляет их суммарную ценность, используя словарь игровых ресурсов.

Условия задачи

СтоимостьПредметы
1камень, палка, тряпка
2дерево, кость, шерсть
3медь, олово, серебро, железо
5бронза, сталь, золото, платина
8рубин, сапфир, алмаз, изумруд
12драконий_зуб, перо_феникса, кристалл_силы
20экскалибур, кольцо_всевластия, философский_камень

💡 Примеры

Тряпка, палка, зОлОто!!!! → 7
Драконий_зуб!!! Кольцо_всевластия... сталь (100) → 37

🛠 Как подойти к решению

  1. Создай словарь, где ключами будут названия предметов (в нижнем регистре), а значениями – их стоимость.
  2. Приведи входную строку к нижнему регистру и очисти её от лишних символов. Можно использовать регулярные выражения или замену знаков препинания на пробелы, после чего разбить строку на слова.
  3. Пройдись по полученному списку слов. Если слово есть в словаре, добавь его стоимость к общей сумме.
  4. Выведи итоговое число.
💻 Решение (нажми, чтобы открыть) 
s = input().lower()

# заменяем все лишние символы на пробелы
for ch in '!,.?()1234567890':
    s = s.replace(ch, ' ')

words = s.split()

prices = {
    'камень': 1, 'палка': 1, 'тряпка': 1,
    'дерево': 2, 'кость': 2, 'шерсть': 2,
    'медь': 3, 'олово': 3, 'серебро': 3, 'железо': 3,
    'бронза': 5, 'сталь': 5, 'золото': 5, 'платина': 5,
    'рубин': 8, 'сапфир': 8, 'алмаз': 8, 'изумруд': 8,
    'драконий_зуб': 12, 'перо_феникса': 12, 'кристалл_силы': 12,
    'экскалибур': 20, 'кольцо_всевластия': 20, 'философский_камень': 20
}

total = 0
for w in words:
    if w in prices:
        total = total + prices[w]

print(total)
✅ Чек-лист для самопроверки:
  • Регистр не влияет на результат
  • Предметы с подчёркиванием распознаются целиком
  • Неизвестные слова и цифры игнорируются
  • Пустой ввод или мусор → вывод 0

🏛️ Задание 2: Римские числа в десятичные

Цель

Напиши конвертер, который принимает строку с римским числом и преобразует её в обычное десятичное целое число.

Условия задачи

Используй базовый словарь соответствий: {'I': 1, 'V': 5, 'X': 10, 'L': 50, 'C': 100, 'D': 500, 'M': 1000}.

💡 Примеры

XIV → 14
MCMXCIV → 1994

🛠 Как подойти к решению

  1. Создай словарь с значениями римских цифр.
  2. Пройдись по строке слева направо. Для каждой цифры сравнивай её значение со значением следующей цифры.
  3. Главное правило: если текущая цифра меньше следующей (например, I перед V), её нужно вычесть из суммы. Во всех остальных случаях – прибавить.
  4. Не забудь обработать последнюю цифру отдельно или выйти за пределы строки аккуратно.
💻 Решение (нажми, чтобы открыть) 
s = input().upper()

rim = {'I': 1, 'V': 5, 'X': 10, 'L': 50, 'C': 100, 'D': 500, 'M': 1000}

total = 0
for i in range(len(s) - 1):
    if rim[s[i]] < rim[s[i + 1]]:
        total = total - rim[s[i]]
    else:
        total = total + rim[s[i]]

# последнюю цифру всегда прибавляем
total = total + rim[s[len(s) - 1]]

print(total)
✅ Чек-лист для самопроверки:
  • Простые числа: III → 3, VI → 6
  • Сложные случаи с вычитанием: IV → 4, IX → 9, XL → 40
  • Длинное число MCMXCIV должно давать 1994
  • Код работает для любой длины корректного римского числа
