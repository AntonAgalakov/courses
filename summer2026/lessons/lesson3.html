🟦 Шаг 1: Инициализация + пустое окно

import pygame pygame.init() screen = pygame.display.set_mode((400, 600)) pygame.display.set_caption("Фрукты") clock = pygame.time.Clock() running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False screen.fill((255, 255, 255)) pygame.display.flip() clock.tick(60) pygame.quit()

✅ Результат: Белое окно 400×600, работает на 60 FPS, закрывается по крестику.

📘 Разбор функций:
  • pygame.init() — запускает все подсистемы. Обязателен первым.
  • set_mode((W, H)) — создаёт окно заданного размера.
  • event.get() — забирает очередь событий (клики, клавиши, закрытие).
  • screen.fill() — стирает предыдущий кадр новым цветом.
  • flip() — выводит отрисованное на экран (двойная буферизация).
  • clock.tick(60) — ограничивает цикл 60 кадрами/сек, экономит CPU.

🟦 Шаг 4: Один падающий фрукт ← +6 строк

import pygame pygame.init() screen = pygame.display.set_mode((400, 600)) clock = pygame.time.Clock() basket = pygame.Rect(170, 550, 60, 20) fruit = pygame.Rect(200, -20, 20, 20) running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False keys = pygame.key.get_pressed() if keys[pygame.K_LEFT]: basket.x -= 5 if keys[pygame.K_RIGHT]: basket.x += 5 basket.clamp_ip(pygame.Rect(0, 0, 400, 600)) fruit.y += 3 if fruit.top > 600: fruit.x = 200 fruit.y = -20 screen.fill((255, 255, 255)) pygame.draw.rect(screen, (139, 69, 19), basket) pygame.draw.rect(screen, (255, 0, 0), fruit) pygame.display.flip() clock.tick(60) pygame.quit()

📘 Разбор функций:

  • .top — свойство Rect с Y верхней грани. Удобнее, чем сырые индексы.

📘 Разбор функций:

🟦 Шаг 6: Множество фруктов ← +10 строк

import pygame import random pygame.init() screen = pygame.display.set_mode((400, 600)) clock = pygame.time.Clock() basket = pygame.Rect(170, 550, 60, 20) fruits = [] spawn_timer = 0 running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False keys = pygame.key.get_pressed() if keys[pygame.K_LEFT]: basket.x -= 5 if keys[pygame.K_RIGHT]: basket.x += 5 basket.clamp_ip(pygame.Rect(0, 0, 400, 600)) spawn_timer += 1 if spawn_timer >= 40: spawn_timer = 0 fruits.append(pygame.Rect(random.randint(20, 380), -20, 20, 20)) new_fruits = [] for f in fruits: f.y += 3 if basket.colliderect(f): pass # пойман elif f.top < 600: new_fruits.append(f) fruits = new_fruits screen.fill((255, 255, 255)) pygame.draw.rect(screen, (139, 69, 19), basket) for f in fruits: pygame.draw.rect(screen, (255, 0, 0), f) pygame.display.flip() clock.tick(60) pygame.quit()

✅ Что изменилось: Один фрукт → список. Таймер спавна и фильтрация упавших объектов.

📘 Разбор функций:
  • random.randint(a, b) — случайное целое число от a до b включительно.
  • new_fruits = [] ... fruits = new_fruits — безопасная фильтрация списка без изменения его во время итерации.

🟦 Шаг 7: Счётчик очков 🎉 ← +4 строки

import pygame import random pygame.init() screen = pygame.display.set_mode((400, 600)) clock = pygame.time.Clock() basket = pygame.Rect(170, 550, 60, 20) fruits = [] spawn_timer = 0 score = 0 font = pygame.font.Font(None, 36) running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False keys = pygame.key.get_pressed() if keys[pygame.K_LEFT]: basket.x -= 5 if keys[pygame.K_RIGHT]: basket.x += 5 basket.clamp_ip(pygame.Rect(0, 0, 400, 600)) spawn_timer += 1 if spawn_timer >= 40: spawn_timer = 0 fruits.append(pygame.Rect(random.randint(20, 380), -20, 20, 20)) new_fruits = [] for f in fruits: f.y += 3 if basket.colliderect(f): score += 1 elif f.top < 600: new_fruits.append(f) fruits = new_fruits screen.fill((255, 255, 255)) pygame.draw.rect(screen, (139, 69, 19), basket) for f in fruits: pygame.draw.rect(screen, (255, 0, 0), f) text = font.render(f"Счёт: {score}", True, (0, 0, 0)) screen.blit(text, (10, 10)) pygame.display.flip() clock.tick(60) pygame.quit()

✅ Что изменилось: Переменная score растёт при поимке. Текст рендерится и накладывается в угол.

📘 Разбор функций:
  • pygame.font.Font(None, 36) — системный шрифт 36pt.
  • font.render(text, antialias, color) — создаёт поверхность с текстом. True = сглаживание.
  • screen.blit(surface, pos) — «приклеивает» поверхность текста на экран.

🟦 Шаг 9: Переход на `blit` и `Rect` спрайтов 📐 ← ~8 строк изменено

import pygame import random pygame.init() screen = pygame.display.set_mode((400, 600)) clock = pygame.time.Clock() basket_img = pygame.image.load("basket.png").convert_alpha() basket_img = pygame.transform.scale(basket_img, (80, 50)) apple_img = pygame.image.load("apple.png").convert_alpha() apple_img = pygame.transform.scale(apple_img, (40, 40)) basket = basket_img.get_rect(center=(200, 550)) fruits = [] spawn_timer = 0 score = 0 font = pygame.font.Font(None, 36) running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False # ... управление ... if random.randint(0, 40) == 0: x = random.randint(20, 380) f_rect = apple_img.get_rect(center=(x, -30)) fruits.append([f_rect, apple_img]) new_fruits = [] for f_rect, f_img in fruits: f_rect.y += 3 if basket.colliderect(f_rect): score += 1 elif f_rect.top < 600: new_fruits.append([f_rect, f_img]) fruits = new_fruits screen.fill((255, 255, 255)) screen.blit(basket_img, basket) for f_rect, f_img in fruits: screen.blit(f_img, f_rect) text = font.render(f"Счёт: {score}", True, (0, 0, 0)) screen.blit(text, (10, 10)) pygame.display.flip() clock.tick(60) pygame.quit()

✅ Что изменилось: Квадраты → картинки. get_rect() даёт точные габариты для коллизий.

📘 Разбор функций:
  • img.get_rect(center=(x,y)) — создаёт Rect размером с картинку, центрирует его.
  • screen.blit(img, rect) — рисует пиксели картинки в позиции rect. Заменяет draw.rect().
  • [f_rect, f_img] — храним пару "позиция + картинка". Двигаем только rect.

🟦 Шаг 10: Разные фрукты + случайный выбор 🍒🍌🍎 ← +5 строк

import pygame import random pygame.init() screen = pygame.display.set_mode((400, 600)) clock = pygame.time.Clock() basket_img = pygame.image.load("basket.png").convert_alpha() basket_img = pygame.transform.scale(basket_img, (80, 50)) apple_img = pygame.image.load("apple.png").convert_alpha() apple_img = pygame.transform.scale(apple_img, (40, 40)) banana_img = pygame.image.load("banana.png").convert_alpha() banana_img = pygame.transform.scale(banana_img, (40, 40)) cherry_img = pygame.image.load("cherry.png").convert_alpha() cherry_img = pygame.transform.scale(cherry_img, (40, 40)) fruit_images = [apple_img, banana_img, cherry_img] basket = basket_img.get_rect(center=(200, 550)) fruits = [] score = 0 font = pygame.font.Font(None, 36) running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False # ... управление ... if random.randint(0, 40) == 0: x = random.randint(20, 380) img = random.choice(fruit_images) f_rect = img.get_rect(center=(x, -30)) fruits.append([f_rect, img]) new_fruits = [] for f_rect, f_img in fruits: f_rect.y += 3 if basket.colliderect(f_rect): score += 1 elif f_rect.top < 600: new_fruits.append([f_rect, f_img]) fruits = new_fruits screen.fill((255, 255, 255)) screen.blit(basket_img, basket) for f_rect, f_img in fruits: screen.blit(f_img, f_rect) text = font.render(f"Счёт: {score}", True, (0, 0, 0)) screen.blit(text, (10, 10)) pygame.display.flip() clock.tick(60) pygame.quit()

✅ Финал: Падают разные фрукты! Логика прежняя, меняется только список картинок.

📘 Разбор функций:
  • random.choice(sequence) — возвращает случайный элемент из списка.
  • fruit_images — централизованный список. Легко добавить новый фрукт: загрузить → масштабировать → добавить.
  • Принцип расширения: Цикл не знает, какая картинка падает, но работает с любой из списка.

Задание 6: Разные очки за разные фрукты

Цель

Яблоко = 1 очко, банан = 2, вишня = 3. Игрок будет стараться ловить более ценные фрукты!

Инструкция

Шаг 1: Создаём словарь «картинка → очки»

После загрузки всех картинок фруктов добавь:

# Словарь: какая картинка сколько очков даёт fruit_scores = { apple_img: 1, banana_img: 2, cherry_img: 3 }

Шаг 2: Используем словарь при поимке

Замени score += 1 на динамическое получение очков:

if basket.colliderect(f_rect): catch_sound.play() score += fruit_scores[f_img] # ← берём очки по картинке
✅ Проверь:
  • Яблоко добавляет +1, банан +2, вишня +3
  • Словарь создан после загрузки всех картинок

⚠️ Частая ошибка

TypeError: unhashable type возникает, если словарь создан до загрузки картинок. Всегда сначала pygame.image.load(), потом словарь!

Задание 7: Штраф за пропущенный фрукт

Цель

Если фрукт упал за нижний край — отнимать 1 очко. Это добавляет элемент риска!

Инструкция

Шаг: Добавляем штраф в цикл обработки

В блоке, где проверяется, упал ли фрукт за экран:

elif f_rect.top > HEIGHT: # фрукт улетел за нижний край score = max(0, score - 1) # ← штраф, но не ниже 0

🔒 Зачем max(0, ...)?
Без этой защиты счёт может уйти в отрицательные значения. Функция max() гарантирует, что результат никогда не будет меньше нуля.

✅ Проверь:
  • Пропущенный фрукт уменьшает счёт на 1
  • Счёт никогда не становится отрицательным

Задание 8: Выход по ESC + сохранение рекорда

Цель

Нажатие ESC завершает игру и сохраняет лучший результат в файл record.txt.

Инструкция

Шаг 1: Загрузка рекорда при старте

После pygame.init() добавь:

record = 0 with open("record.txt", "r") as f: record = int(f.read().strip())

Шаг 2: Обработка нажатия ESC

В цикле for event in pygame.event.get(): добавь:

if event.type == pygame.KEYDOWN: if event.key == pygame.K_ESCAPE: if score > record: with open("record.txt", "w") as f: f.write(str(score)) running = False # выход из главного цикла
⚠️ Важно:
  • Проверка ESC должна быть в event.get(), а не в key.get_pressed()
  • Используй with open(...) — это автоматически закроет файл даже при ошибке
✅ Проверь:
